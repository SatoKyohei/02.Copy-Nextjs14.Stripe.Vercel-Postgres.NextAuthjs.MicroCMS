## 実行したコマンド（順不同）
- npm i next-auth
- npm i @next-auth/prisma-adapter
- npm i -g vercel
- npm i @prisma/client
- npm i @vercel/postgres
- npm i microcms-js-sdk
- npx prisma init
- npx prisma migrate dev --name init
- npx prisma migrate dev
- npm i stripe
- npm i react-spinners
